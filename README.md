## Project Overview

This project analyzes New York City taxi trip data using exploratory data analysis (EDA) to uncover temporal, spatial, and economic patterns in urban transportation. Through visualizations and summary statistics, we explore trip volumes, distances, fare distributions, tipping behavior, and passenger counts to build intuition about large-scale mobility data and identify meaningful trends.

## Key Features

- **Temporal Analysis**: Pickup and drop-off time distributions by hour, day, and month
- **Trip Characteristics**: Distance and duration distributions
- **Economic Patterns**: Fare amounts, tip amounts, and total cost behavior
- **Passenger Analysis**: Passenger count frequency patterns
- **Payment Insights**: Payment type usage analysis
- **Data Quality**: Identification of outliers and anomalous trips

## Project Context

This is a self-created project designed for learning and development purposes, without an external sponsor or client.

## Challenges

- Large dataset requiring filtering and sampling for efficient analysis
- Extreme outliers in distance and fare data requiring careful handling
- Limited geographic interpretation without map-based visualization tools
